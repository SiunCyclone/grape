Grape
======

Grape is a Cross-Platform 3D Library for D.

### Status
Very Alpha

![](http://i.imgur.com/VOfA957.png)

### Building

##### Requirement
* [Dub](https://github.com/D-Programming-Language/dub)

##### Procedure

Just add below code to your dub.json

```
{
    "dependencies": {
        "grape": "~master"
    },
    
    "targetType": "executable"
}
```
